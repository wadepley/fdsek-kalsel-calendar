🔥 Konfigurasi Firebase
1Buat Project Firebase
Buka console.firebase.google.com → Add project → beri nama (misal: fdsek-kalsel) → Continue → Disable Analytics → Create project
2Aktifkan Realtime Database
Di sidebar kiri → Build → Realtime Database → Create Database → pilih region Singapore (asia-southeast1) → Start in test mode → Enable.

Setelah database terbuat, klik tab Rules → ganti dengan rules berikut → Publish:
{ "rules": { ".read": true, ".write": true } }
3Salin Database URL
Di tab Data, lihat URL di bagian atas — contoh:
https://fdsek-kalsel-default-rtdb.asia-southeast1.firebasedatabase.app
Salin URL tersebut dan paste di bawah ini:
💡 Paste URL lengkap dari tab Data Firebase (jangan ada "/" di akhir)
4Tambahkan Domain (jika perlu)
Jika Test Koneksi gagal meski URL & Rules sudah benar, buka Firebase Console → Authentication → Settings → Authorized domains → klik Add domain → tambahkan domain tempat kalender di-publish (misal: claude.site atau domain Anda).
